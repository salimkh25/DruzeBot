# 🤖 בוט קבוצת החיילים הדרוזים

## מה הבוט עושה
- ✅ שאלון כניסה מלא עם 6 שאלות
- ✅ קבלת תמונת תעודה
- ✅ שליחה למנהל לאישור/דחייה
- ✅ מספור חברים אוטומטי (#001, #002...)
- ✅ הודעת ברכה + הנחיות לכל חבר חדש
- ✅ מערכת אזהרות (פעמיים = הוצאה)
- ✅ Cooldown 24 שעות אחרי דחייה

---

## הגדרה ראשונית (10 דקות)

### שלב 1 – יצירת הבוט
1. פתח טלגרם וחפש **@BotFather**
2. שלח `/newbot`
3. בחר שם ושם משתמש לבוט
4. תקבל **TOKEN** – שמור אותו!

### שלב 2 – מציאת ה-IDs שלך
- **ADMIN_ID** (המזהה שלך): חפש @userinfobot בטלגרם ושלח לו הודעה
- **GROUP_ID** (מזהה הקבוצה): הוסף את @userinfobot לקבוצה, הוא יגיד את ה-ID

### שלב 3 – העלאה ל-Railway
1. פתח חשבון ב-https://railway.app (חינמי)
2. לחץ "New Project" → "Deploy from GitHub"
3. העלה את הקבצים (bot.py + requirements.txt)
4. לחץ על הפרויקט → "Variables" → הוסף:
   ```
   BOT_TOKEN = הטוקן שקיבלת מ-BotFather
   ADMIN_ID  = המזהה שלך
   GROUP_ID  = מזהה הקבוצה (עם מינוס לפני, למשל: -1001234567890)
   ```
5. לחץ "Deploy" – הבוט חי! 🎉

## הגדרת הקבוצה
- הגדר את הקבוצה כ**פרטית**
- הוסף את הבוט לקבוצה כ**מנהל** (עם הרשאת הוצאת חברים)
- בהגדרות הקבוצה: **"Who can add members"** → "Admins only"
- כשמישהו רוצה להצטרף, שלח לו את הלינק לבוט: t.me/שם_הבוט
